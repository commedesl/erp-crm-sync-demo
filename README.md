# ERP CRM Sync System

This project simulates real-world data synchronization between ERP and CRM systems.

## Features
- Separate ERP and CRM interfaces
- Centralized project number generation (no duplication)
- Data synchronization between systems
- Difference checking before sync
- Demo data reset

## Tech
- Google Apps Script
- Google Sheets
- HTML / JavaScript

## Concept
- Avoid duplicate project numbers using central sequence control
- Maintain consistency between multiple systems
- Reduce manual data update errors

## Demo Flow
1. Create project in ERP or CRM
2. System auto-generates project number
3. Sync data between systems
4. Check differences before and after sync
